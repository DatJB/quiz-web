# HỆ THỐNG THI TRẮC NGHIỆM ONLINE

Hệ thống thi trắc nghiệm dành cho sinh viên và quản trị viên, được xây dựng bằng:

- HTML5
- CSS3
- Tailwind CSS
- JavaScript 
- LocalStorage 

## TÍNH NĂNG CHÍNH

### Xác thực người dùng

- Đăng nhập sinh viên
- Đăng ký sinh viên
- Đăng nhập Admin
- Phân quyền Student / Admin
- Lưu session bằng localStorage

###  Phần Sinh Viên

- Xem danh sách kỳ thi
- Tìm kiếm & lọc kỳ thi
- Làm bài thi trắc nghiệm
- Đồng hồ đếm ngược
- Xem kết quả chi tiết
- Xem lịch sử thi
- Cập nhật thông tin cá nhân
- Đổi mật khẩu

### Phần Admin

- Dashboard thống kê tổng quan
- Quản lý kỳ thi (Thêm / Sửa / Xóa)
- Tạo đề thi mới
- Xem kết quả sinh viên
- Quản lý tài khoản sinh viên
- Thống kê & báo cáo
